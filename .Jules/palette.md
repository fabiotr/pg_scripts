## 2026-03-09 - Standardizing Report Output to English
**Learning:** In a predominantly English codebase, maintaining language consistency in reports (even those generated via CLI tools like psql) is crucial for professional presentation and accessibility. Localized strings in report headers or column aliases can disrupt the user flow for international users.
**Action:** Audit report generation scripts for language consistency and ensure all user-facing output adheres to a single standard language (English in this case).
