## 2025-05-15 - [Consistency in Script Naming and Report Headers]
**Learning:** In technical script repositories, consistency between filenames, internal references, and report headers is crucial for trust and discoverability. Using plural forms for multi-item sections (e.g., 'Security Labels' instead of 'Security Label') aligns with user expectations for inventory reports.
**Action:** Always audit filenames and output headers for spelling and pluralization consistency when adding or modifying reporting modules.
